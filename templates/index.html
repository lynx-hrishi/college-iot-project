Hardware
Components Used
🧠
Arduino Uno
Microcontroller · Brain
The central processing unit of the project. It reads sensor data, applies classification logic, and controls output indicators (LEDs/servo motor). Runs on 5V and operates at 16 MHz.
ATmega328P 14 Digital I/O 6 Analog In 5V / 16MHz
📡
HC-SR04
Ultrasonic Sensor · Proximity Gate
Emits ultrasonic pulses and measures echo return time to calculate object distance. Used as the entry trigger — if an object enters within the defined minimum distance, the classification pipeline activates.
Range: 2–400 cm Trigger + Echo 5V VCC 40kHz
🌧
FC-37 Rain Sensor
Moisture Detector · Wet Classification
Detects moisture or water content on the waste surface via electrical conductivity between its sensing traces. Outputs both analog voltage (moisture level) and a digital HIGH/LOW signal relative to a set threshold.
Analog + Digital Out 3.3V – 5V LM393 Comparator Adjustable Threshold
🧲
A3144 Hall Sensor
Magnetic Field Detector · Metal Classification
A unipolar Hall effect sensor that detects the presence of a magnetic field from ferrous or magnetic waste objects. Outputs LOW when a south-pole magnetic field is present, making it ideal for detecting metal items near a magnet mounted on the bin.
Digital Output 4.5V – 24V Unipolar Open Collector
💡
Output Indicators
LEDs / Servo · Visual Feedback
RGB LEDs or individual color LEDs (Blue=Wet, Yellow=Dry, Silver/White=Metal) provide instant visual feedback on classification result. Optionally, a servo motor can physically route waste into the correct bin compartment.
3× LEDs 220Ω Resistors Servo (Optional) 5V Logic
